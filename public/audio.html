Commentary Checklist

1. Open the listener Load overlay.html in OBS as a Browser Source or open it in a browser tab while you test.
2. Run Mic Check Confirm the correct microphone, watch the level meter, and use Monitor if you are on headphones.
3. Start Commentary The Overlay linked badge should turn green after the overlay answers the WebRTC offer.
4. Trust, then go live If the badges stay green and the event log shows the link completed, you are in good shape to stay on-air.
